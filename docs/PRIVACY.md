- **Учётная запись:** имя пользователя, хеш пароля (Argon2id), публичный ключ для E2EE (как передаётся клиентом).

- **Технические данные:** IP, логи безопасности, метаданные запросов — в объёме, необходимом для работы и защиты от злоупотреблений.
